Code Battle

Code Battle is a web application that allows users to practice their coding skills. One of the core aspects of the application is that users can challenge each other to code battles in real time to see who can complete challenges in the quickest time. User code is executed on a Docker instance and Socket.io is utilized for real time interactivity.

Password Rules

Password Rules is a website that allows users to submit password requirements for websites that they use. The site utilizes Node.js/Express for the server and MongoDB ad the database. The front end is built using React/Redux.

Running Man

Running Man is a Javascript game built using the Phaser Game Framework. The main focus of the game is the ever changing gamplay. It starts out as a simple running game but quickly evolves. The art assets were all created by hand using Gimp.
