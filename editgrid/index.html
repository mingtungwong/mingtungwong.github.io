The grid is made using a collection of divs. Initially the user creates a grid by specifying the row and column amounts. A default value for each grid can also be supplied otherwise the deault value is "empty". After creating the grid, users can modify the values by specifying the row, column and new value using the form on the bottom. Users can also click on each grid cell to prefill the row and column on the form. Clicking on a grid cell will highlight it. Filling out the grid cell location on the form will also highlight the corresponding cell on the grid. Creating a grid after having already created one will delete the old grid and create a new one.
